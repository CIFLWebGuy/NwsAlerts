      NwsAlertApi Namespace

      Contains the classes that are used to make Web API calls to retrieve NWS alert data.

      Classes

      Address Defines an address.
      Alert Defines the properties for an NWS Alert
      AlertFilters Defines properties used to filter query results.
      AlertParameters Defines the parameters for an alert.
      AlertReference Defines an alert reference.
      AlertResponse Defines the response for the /Alert endpoint.
      ApiClient Defines the properties and methods used to access the NWS Alert API.
      GeoCode Contains SAME and UGC lists
      GeographicArea Defines the properties used to define a geographic area in API calls.
      LdContext Defines the LD context
      NwsOffice Defines the properties for an NWS office.
      ObservationStation Defines the properties of an observation station.
      Zone Defines the properties of a NWS zone.
      ZoneResponse Defines the response for the /Zone endpoint.
